Om Bolycka

Vår vision

Bolycka skapades som ett gymnasiearbete på Härnösands gymnasium med målet att göra öppen data mer tillgänglig och användarvänlig för alla. Vi tror att denna typ av information kan hjälpa människor att fatta mer välgrundade beslut om var de vill bo.

Hur det fungerar

Vår AI-analyserar hundratals datapunkter från offentliga källor som:

  • SCB (Statistiska centralbyrån)
  • Sveriges dataportal
  • SMHI (väderdata)
  • Kommuners egna databaser
  • Trafikverkets kollektivtrafikdata

Genom att ställa några enkla frågor om dina preferenser kan vi matcha dig med den ort som bäst passar dina behov.
